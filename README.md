
### **项目概述**
基于RV1126嵌入式平台，实现低延迟的视觉处理流水线：  
**V4L2摄像头采集 → YUV原始数据直传 → 视觉模型推理(RKNN) → GPU加速渲染(OpenGL ES) → Qt上位机显示**。  
模块化设计，支持多模型动态加载(如异常细胞检测、行星识别等)，适用于医疗、天文、工业质检等场景。

---

### **功能特性**
- **直取流水线**: 通过`V4L2`直接获取YUV数据，避免内存复制，降低延迟。  
- **多模型支持**: 动态加载RKNN模型，支持分类、检测、分割任务。  
- **GPU加速**: 使用OpenGL ES实现YUV→RGB转换与渲染，释放CPU算力。  
- **Qt交互界面**: 实时显示处理结果，支持标注、参数配置与数据导出。
